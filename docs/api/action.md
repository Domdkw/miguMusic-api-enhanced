# 活动接口

## 呢喃-歌曲列表

**接口地址**: `/ninan/song`  
**请求方法**: `GET`

### 参数说明

无
### 请求示例

```
/ninan/song
```

---

## 呢喃-日期歌曲

**接口地址**: `/ninan/date`  
**请求方法**: `GET`

### 参数说明

| 参数名 | 类型 | 必填 | 说明 | 示例 |
| ------- | ------ | ------ | ------ | ------ |
| date | string | 是 | 日期 | 20260713 |

!> 日期格式为 YYYYMMDD，有些时间没有呢喃歌曲，会返回默认歌曲

### 请求示例

```
/ninan/date?date=20260713
```
