# 专辑接口

## 专辑详情

**接口地址**: `/album/info`  
**请求方法**: `GET`

### 参数说明

| 参数名 | 类型 | 必填 | 说明 | 示例 |
| ------- | ------ | ------ | ------ | ------ |
| albumId | string | 是 | 专辑 ID | 1138608258 |

### 请求示例

```
/album/info?albumId=1138608258
```

---

## 专辑歌曲列表

**接口地址**: `/album/song`  
**请求方法**: `GET`

### 参数说明

| 参数名 | 类型 | 必填 | 说明 | 示例 |
| ------- | ------ | ------ | ------ | ------ |
| albumId | string | 是 | 专辑 ID | 1138608258 |
| page | number | 否 | 页码，默认 1 | 1 |

### 请求示例

```
/album/song?albumId=1138608258&page=1
```
