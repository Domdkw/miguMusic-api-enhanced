# 排行榜接口

## 排行榜栏目

**接口地址**: `/rank/index`  
**请求方法**: `GET`

### 参数说明

无需参数

### 请求示例

```
/rank/index
```

---

## 排行榜内容

**接口地址**: `/rank/info`  
**请求方法**: `GET`

### 参数说明

| 参数名 | 类型 | 必填 | 说明 | 示例 |
| ------- | ------ | ------ | ------ | ------ |
| rankId | string | 是 | 排行榜 ID | 27553319 |
| page | number | 否 | 页码，默认 1 | 1 |

### 请求示例

```
/rank/info?rankId=27553319&page=1
```
