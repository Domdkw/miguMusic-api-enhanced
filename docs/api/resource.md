# 资源接口

## 资源信息

**接口地址**: `/resourceinfo`  
**请求方法**: `GET`

### 参数说明

| 参数名 | 类型 | 必填 | 说明 | 示例 |
| ------- | ------ | ------ | ------ | ------ |
| resourceId | string | 是 | 资源 ID（支持竖线分隔多 ID） | 1131254786\|1000002956 |
| resourceType | string | 是 | 资源类型 | 2002 |

### 请求示例

```
/resourceinfo?resourceId=1131254786|1000002956&resourceType=2002
```

## 检查歌曲是否可听

**接口地址**: `/can-listen`  
**请求方法**: `POST`

### 参数说明

请求体为 JSON 格式

| 参数名 | 类型 | 必填 | 说明 | 示例 |
| ------- | ------ | ------ | ------ | ------ |
| contentIds | string | 是 | Content ID 列表（逗号分隔） | 600902000000173077,600929000000900351 |

### 请求示例

```
/can-listen
```

请求体：

```json
{
  "contentIds": "600902000000173077,600929000000900351,600908000009237414"
}
```

## 歌曲播放次数

**接口地址**: `/opNum`  
**请求方法**: `GET`

### 参数说明

| 参数名 | 类型 | 必填 | 说明 | 示例 |
| ------- | ------ | ------ | ------ | ------ |
| opType | string | 是 | 操作类型 | play |
| resourceType | string | 是 | 资源类型 | 2021 |
| ids | string | 是 | 资源 ID 列表（逗号分隔） | 233851696,234592877 |

### 请求示例

```
/opNum?opType=play&resourceType=2021&ids=233851696,234592877
```

## 票务信息

**接口地址**: `/ticket`  
**请求方法**: `GET`

### 参数说明

| 参数名 | 类型 | 必填 | 说明 | 示例 |
| ------- | ------ | ------ | ------ | ------ |
| page | number | 否 | 页码，默认 1 | 1 |
| pageCount | number | 否 | 每页数量，默认 10 | 10 |

### 请求示例

```
/ticket?page=1&pageCount=10
```
