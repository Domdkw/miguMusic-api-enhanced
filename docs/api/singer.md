# 歌手接口

## 歌手信息

**接口地址**: `/singer/index`  
**请求方法**: `GET`

### 参数说明

| 参数名 | 类型 | 必填 | 说明 | 示例 |
| ------- | ------ | ------ | ------ | ------ |
| singerId | string | 是 | 歌手 ID | 953362 |

### 请求示例

```
/singer/index?singerId=953362
```

---

## 歌手歌曲列表

**接口地址**: `/singer/song`  
**请求方法**: `GET`

### 参数说明

| 参数名 | 类型 | 必填 | 说明 | 示例 |
| ------- | ------ | ------ | ------ | ------ |
| singerId | string | 是 | 歌手 ID | 953362 |
| page | number | 否 | 页码，默认 1 | 1 |

### 请求示例

```
/singer/song?singerId=953362&page=1
```
