# 短视频&视频彩铃接口

## 搜索短视频

**接口地址**: `/svideo/search`  
**请求方法**: `GET`

### 参数说明

| 参数名 | 类型 | 必填 | 说明 | 示例 |
| ------- | ------ | ------ | ------ | ------ |
| text | string | 是 | 搜索关键词 | APT. |
| page | number | 否 | 页码，默认 1 | 1 |

### 请求示例

```
/svideo/search?text=APT&page=1
```

## 短视频URL

**接口地址**: `/svideo/url`  
**请求方法**: `GET`

### 参数说明

| 参数名 | 类型 | 必填 | 说明 | 示例 |
| ------- | ------ | ------ | ------ | ------ |
| contentIds | string | 是 | 短视频 ID | 600985A00005943149,600985A00004881354 |
| formatType | string | 否 | 质量 (默认 HQ) | PQ |

逗号分隔的短视频 ID 列表

### 请求示例

```
/svideo/url?contentIds=600985A00005943149,600985A00004881354&formatType=SQ
```

## 短视频用户作品

**接口地址**: `/svideo/userContent`  
**请求方法**: `GET`

### 参数说明

| 参数名 | 类型 | 必填 | 说明 | 示例 |
| ------- | ------ | ------ | ------ | ------ |
| videoUserId | string | 是 | 用户 ID | 10268521 |
| nextPageId | number | 否 | 页码编号 |  |

?> 请求第一页时页码编号为空，后续页码编号为返回值的 nextPageId。

### 请求示例

```
/svideo/userContent?videoUserId=10268521&nextPageId=685797c386a1c849d0a4a45e
```

### 响应示例

```json
{
  "code": "000000",
  "info": "操作成功",
  "data": {
    "content": [{},{}],
    "hasNextPage": true,
    "nextPageId": "685797c386a1c849d0a4a45e",
    "total": 1042,
    "hasPrePage": false,
    "videoAssembleList": [{},{}]
  }
}
```
